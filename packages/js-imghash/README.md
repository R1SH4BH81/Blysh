# imghash

Image hashing and decoding (BlurHash-like) in TypeScript and React.

## Installation

```bash
npm install imghash
```

## Usage

### Encoding

You can encode directly from an `HTMLImageElement`:

```typescript
import { encodeImage } from 'imghash';

const img = document.querySelector('img');
const hash = encodeImage(img, 4, 3);
```

Or from raw pixel data (`Uint8ClampedArray`):

```typescript
import { encode } from 'imghash';

const pixels = ctx.getImageData(0, 0, width, height).data;
const hash = encode(pixels, width, height, 4, 3);
```

### Decoding

```typescript
import { decode } from 'imghash';

const pixels = decode(hash, width, height);
```

### Simple Usage (Progressive Loading)

The easiest way to use this is with the `ImgHash` component. It automatically shows the blurred placeholder until the real image loads:

```tsx
import { ImgHash } from 'imghash';

const App = () => (
  <ImgHash 
    hash="L6PZfS_u.AyD_N%g9G%M_3WX9G?b" 
    src="https://example.com/high-quality-image.jpg"
    className="w-full h-64 rounded-xl"
  />
);
```

### Advanced Usage (Canvas Only)

If you're not using React, you can use the low-level `decode` function to get raw pixel data:

```typescript
import { decode } from 'imghash';

const pixels = decode(hash, 32, 32);
const imageData = new ImageData(pixels, 32, 32);
ctx.putImageData(imageData, 0, 0);
```

### Compatibility

The hash generated by the `imghash` Python package is **fully compatible** with this JS package. You can encode on your backend and decode on your frontend.
