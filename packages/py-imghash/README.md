# imghash

A simple image hashing library (BlurHash-like) in Python.

## Installation

```bash
pip install imghash
```

## Usage

### Encoding

```python
from PIL import Image
from imghash import encode

image = Image.open("path/to/image.jpg")
hash = encode(image, components_x=4, components_y=3)
print(hash)
```

### Decoding

The decoding logic is currently implemented in TypeScript/JS for frontend rendering. You can use the utilities in `imghash.algo` for lower-level operations.

The hash generated by this Python module is **fully compatible** with the `imghash` JS package. You can encode on your backend and decode on your frontend.
